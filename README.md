## 구현 기능


### 기능 1
- Flex, Grid를 통한 반응형 웹페이지

### 기능 2
- React Query를 통해 데이터 관리하기

### 기능 3
- Cloudnary이용해 이미지 업로드하기

### 기능 4
- FireBase api 사용하여 auth, admin, realtimedate 등 사용
